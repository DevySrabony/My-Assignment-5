Question-1: What are the different ways to select an element in the DOM?

There are several ways to grab an element from the DOM

      getElementById('id')
      getElementsByClassName('class')
      getElementsByTagName('tag')
      querySelector('selector')
      querySelectorAll('selector')
Question-2: What is the difference between innerHTML, innerText, and textContent ?

`innerHTML` gets or sets content, including HTML tags. `innerText` retrieves only the **visible** text, ignoring hidden elements. `textContent` gets all text, including hidden ones, but treats HTML as plain text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where I **attach a single event listener** to a parent element instead of multiple child elements. This works because events **bubble up** from the target element to its ancestors. It helps improve performance and makes handling dynamic elements easier.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts from the element that was clicked and moves **upward** through its parent elements. For example, if i click a button inside a div, the event first happens on the button, then "bubbles up" to the div, then to the body, and so on. This helps in handling events at a higher level instead of adding listeners to every small element.

Question-5: How do you create, add, and remove elements using JavaScript?
To create an element in JavaScript, I use document.createElement('tagName'). To add it to the page, I can use .appendChild() or .append() on a parent element. To remove an element, I use .remove() on the element itself.
